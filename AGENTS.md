# AGENTS.md - DnevnikFlutter (Школьный Дневник 10А)

## 📌 Мета-информация
- **Name:** DnevnikFlutter / dnevnik_app
- **Goal:** Мобильное Flutter-приложение для расписания уроков и домашних заданий класса 10А
- **Audience:** Ученики и учителя класса 10А
- **Phase:** prototype
- **Tech Stack:** Flutter 3.x, Dart SDK ^3.8.0, Riverpod 3 (state management), go_router 17 (навигация), Firestore (backend), freezed + json_serializable (модели), flutter_secure_storage, flutter_dotenv
