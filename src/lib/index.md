# Карта исходного кода (Code Map)

Приложение построено на базе Flutter 3.x с использованием **Riverpod 3** (стейт-менеджмент) и **go_router** (навигация).

## Архитектура папок
- `data/` — модели данных (через `freezed`) и репозитории для работы с Firestore.
- `providers/` — Riverpod-провайдеры (бизнес-логика, контроллеры состояния).
- `screens/` — экраны приложения (UI). Каждый экран обычно привязан к своему провайдеру.
- `widgets/` — переиспользуемые UI-компоненты (кнопки, карточки, инпуты). Не дублируй код, ищи существующие виджеты!
- `router/` — конфигурация навигации через `go_router`.
- `theme/` — константы темы, цвета (Sahara Warm Minimalism) и стили текста. Никакого хардкода цветов в экранах!
- `utils/` — вспомогательные функции и утилиты.
