# Карта проекта DnevnikFlutter (Root Map)

Добро пожаловать в проект Школьный Дневник. Этот файл — ваша отправная точка.

## Навигация по проекту:
- `AGENTS.md` — глобальные правила работы агентов (золотые правила, что можно и нельзя делать).
- `docs/` — память проекта, документация, архитектура и логи сессий. Обязательно прочитай `docs/index.md` для понимания, где что лежит!
- `src/lib/` — исходный код Flutter приложения. Карта кода находится в `src/lib/index.md`.
- `logs/sessions/` — история работы, логи каждой сессии и выводы "Lessons Learned".
- `.agents/skills/` — локальные ИИ-навыки, автоматизирующие рабочую рутину.

Если ты только начинаешь сессию, обязательно прочитай `AGENTS.md` и последний лог в `logs/sessions/`.
