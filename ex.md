```markdown
# Архитектурные решения

## 📋 Формат

```
## YYYY-MM-DD - [Название решения]
- **Status:** proposed | accepted | rejected | replaced
- **Decision:** [Что решили делать]
- **Why:** [Почему этот вариант лучше]
- **Consequences:** [Что становится проще/сложнее]
- **Rollback:** [Как откатиться если понадобится]
```

## Примеры

### 2025-01-15 - SQLite вместо PostgreSQL
- **Status:** accepted
- **Decision:** MVP на SQLite
- **Why:** Один юзер, проще деплой, дешевле
- **Consequences:** Нужна миграция на PG когда multi-user
- **Rollback:** Переписать Prisma schema

### 2025-01-16 - Shadcn/ui вместо Material-UI
- **Status:** accepted
- **Decision:** Дизайн система Shadcn
- **Why:** Минималистично, настраивается, лёгкий
- **Consequences:** Все компоненты через Shadcn
```

##  docs/RECOVERY.md (СКОПИРУЙ КАК ЕСТЬ)

```markdown
# Recovery Guide

Если потеряешь историю чата, используй этот процесс.

## 📋 Порядок восстановления
1. Прочитай AGENTS.md
2. Прочитай docs/PROJECT_CONTEXT.md
3. Прочитай docs/DECISIONS.md
4. Прочитай docs/CHANGELOG.md
5. Прочитай 3 последних файла в logs/sessions/

## 🔄 Что нужно восстановить
- Что это за проект?
- На каком этапе находится?
- Какие решения приняты?
- Что последнее было сделано?
- Какие файлы были изменены?
- Какие есть открытые задачи?
- Какие есть риски?

## 🚀 Recovery Prompt (скопируй целиком в новый чат)

```
# Recovery Session

Я потерял историю чата с проектом.
Помоги восстановить контекст из локальных файлов.

## Файлы для чтения
1. AGENTS.md — правила для агентов
2. docs/PROJECT_CONTEXT.md — контекст проекта
3. docs/DECISIONS.md — архитектурные решения
4. docs/CHANGELOG.md — история изменений
5. logs/sessions/ — логи последних 2-3 дней

## Что нужно восстановить
1. **Карта проекта:** что это? зачем? на каком этапе?
2. **Последнее состояние:** что сделано? какие файлы были изменены?
3. **Открытые задачи:** что дальше?
4. **Риски:** что может сломаться?
5. **Решения:** какие архитектурные решения приняты?

## Инструкции
- Ничего не меняй, только ответь
- Будь краток, но полнота важна
- Если какой-то файл не понял, скажи

Готов? Начни читать файлы.
