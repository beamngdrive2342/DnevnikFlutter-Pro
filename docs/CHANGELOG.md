# Changelog — DnevnikFlutter

## [2026-06-30]

### Fixed
- **Vercel Proxy:** Исправлена критическая ошибка сборки прокси-сервера Gemini API на Vercel (возвращал 404 NOT_FOUND при любых запросах к API). В настройках проекта Vercel `rootDirectory` изменен с `.` на `src/vercel-proxy`, благодаря чему автоматический Git-билд теперь корректно находит и компилирует Edge-функцию `gemini.js` по пути `api/api/gemini.js`.

---

## [2026-06-27]

### Added
- Созданы экспериментальные виджеты `GeometrySolution1` и `GeometrySolution8` (в папке `widgets/geometry/`) для тестирования отрисовки чертежей через `CustomPaint`.

### Changed
- Обновлены правила агента (`AGENTS.md`) и архитектурные решения (`DECISIONS.md`): принята стратегия парсинга текста ГДЗ в Markdown/LaTeX, а ручная отрисовка чертежей на Canvas запрещена из-за немасштабируемости для 3400+ задач.

---

## [2026-06-25]

### Fixed
- Изображения домашних заданий (ДЗ) теперь сохраняются в оффлайн-кэш телефона (переписан `NetworkPhoto` с использованием `CachedNetworkImage`).
- Исправлена проблема с прыгающим скроллом при просмотре ГДЗ (`ListView` заменен на `SingleChildScrollView`).
- Оптимизирована загрузка ГДЗ: количество изображений для каждого номера теперь кэшируется локально, избегая лишних `HEAD` запросов к серверу при повторном открытии ответа.

### Removed
- **(Отменено)** Попытка внедрения push-уведомлений через FCM из-за сложностей настройки Vercel Node.js.
- **(Отменено)** Эксперимент с использованием `flutter_markdown` + `flutter_math_fork` + `flutter_svg` для рендеринга собственных решений ГДЗ. Создание геометрии с перспективой через сырой SVG код оказалось слишком неточным и было удалено.

### Reverted
- Экспериментальная интеграция Push-уведомлений (FCM) и Vercel-бэкенда для их рассылки. Все изменения отменены из-за сложностей настройки серверного окружения на Vercel (ошибка импортов ESM/CommonJS).

---

## [2026-06-24]

### Added
- Автоматическая подгрузка готовых ответов (ГДЗ) по Геометрии и Алгебре при просмотре домашнего задания.
- Внедрение ИИ-распознавания фотографий с доски при быстром добавлении домашнего задания (Gemini Vision API). ИИ научился находить номера задач на фото и связывать их с предметом.
- На экране ученика (`LessonCard`) добавлена кнопка "Открыть ответы", которая вызывает непрозрачное модальное окно с фотографиями решений (фотографии загружаются из сети).

### Changed
- В модель `HomeworkItem` добавлено поле `textbookNumbers`.
- Обновлен системный prompt ИИ в `ai_service.dart`: теперь он игнорирует скобки и буквы при парсинге номеров из текста/фото.
- **Миграция ресурсов:** Готовые ответы (около 3400 картинок по Геометрии и Алгебре) перенесены в публичный репозиторий GitHub.
- Изображения теперь загружаются через `cached_network_image` для обеспечения надежного офлайн-кеширования.

### Fixed
- **Офлайн режим:** Исправлен баг, из-за которого происходил разлогин при запуске приложения без интернета. Добавлено локальное сохранение документа класса в `SharedPreferences`.
- **Офлайн режим:** Исправлено отображение нескольких фотографий решения в офлайне (теперь количество фото для каждого номера кешируется локально, что позволяет `cached_network_image` выводить их без сетевых `http.head` проверок).
