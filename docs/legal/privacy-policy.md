# Политика конфиденциальности — Школьный Дневник

**Дата последнего обновления:** 6 июля 2026 года

## 1. Кто мы

«Школьный Дневник» — мобильное приложение для школьников, предназначенное для просмотра расписания уроков, управления домашними заданиями и получения помощи в учёбе через встроенного AI-ассистента.

Приложение разработано как учебный проект.

## 2. Какие данные мы собираем

### 2.1 Данные администратора класса
- **Email-адрес** — используется как учётные данные для входа в систему (Firebase Authentication)
- **Название класса и школы** — указываются при создании класса

### 2.2 Данные учеников
- **Анонимный идентификатор** — Firebase Auth создаёт уникальный анонимный ID при вступлении в класс; никакие персональные данные не запрашиваются

### 2.3 Данные о домашних заданиях
- **Текст заданий, дедлайны, прикреплённые изображения** — сохраняются в облаке (Firestore) и видны всем участникам класса

### 2.4 AI-запросы
- **Текст вопросов к AI-ассистенту и прикреплённые изображения** — передаются на сервер для обработки через Google Gemini API. Мы не сохраняем историю чата на сервере — она хранится только локально на вашем устройстве.

### 2.5 Локальные данные
- **Кэш расписания и домашних заданий** — сохраняется локально на устройстве в `SharedPreferences` для работы без интернета
- **Токен авторизации** — хранится в зашифрованном хранилище устройства (`Keystore` на Android)
- **Настройки темы** — сохраняются локально

## 3. Как мы используем данные

- Для авторизации пользователей и разграничения доступа
- Для синхронизации расписания и домашних заданий между участниками класса
- Для обработки запросов к AI-ассистенту
- Мы **не продаём** ваши данные третьим лицам
- Мы **не используем** данные для рекламы

## 4. Где хранятся данные

- **Облако:** Google Cloud (Firebase Firestore, Firebase Authentication) — серверы в США/Европе
- **AI-обработка:** Google Gemini API через наш прокси-сервер (Vercel)
- **Устройство:** кэш и токены авторизации — только на вашем устройстве

## 5. Как удалить данные

Вы можете удалить все ваши данные прямо из приложения:

1. Откройте **Настройки** (иконка шестерёнки)
2. Прокрутите вниз до секции **«Аккаунт»**
3. Нажмите **«Удалить аккаунт и данные»**
4. Подтвердите действие

**Для администратора:** удаление аккаунта удаляет весь класс, включая все домашние задания и данные участников.

**Для ученика:** удаление аккаунта удаляет вашу запись участника класса и анонимный ID.

## 6. Дети и возрастное ограничение

Приложение предназначено для использования школьниками. Мы не собираем дополнительные данные, которые выходили бы за рамки необходимых для работы приложения. Родители/опекуны могут связаться с нами для удаления данных своего ребёнка.

## 7. Безопасность данных

- Токены авторизации хранятся в зашифрованном хранилище устройства
- Соединение с серверами использует HTTPS
- Доступ к данным класса ограничен через правила безопасности Firebase
