# Карта документации (Docs Map)

В этой папке хранится вся накопленная память проекта DnevnikFlutter. Мы используем структуру: Raw -> Wiki -> Outputs.

## Структура
- `docs/wiki/` — **Чистовики и База Знаний**
  - `PROJECT_CONTEXT.md` — общая концепция, бизнес-логика и модели данных.
  - `DECISIONS.md` — принятые архитектурные решения.
  - `AGENTS_ROLES.md` — роли в проекте.
  - `AGENTS_KNOWLEDGE.md` — (будет создан при необходимости) накопленные инсайты и уроки из "Lessons Learned".
- `docs/raw/` — **Сырые данные**
  - Дампы, транскрипции, идеи для брейнштормов, логи ошибок.
- `docs/outputs/` — **Результаты**
  - `CHANGELOG.md` — история изменений кода.
  - Сборки, ссылки на `.apk`.
