# Начать сессию (start-session)

Когда пользователь просит начать новую сессию (или вызывает этот навык), выполни следующие шаги:

## Обязательные действия:
Обязательно прочитай (используя инструменты для чтения файлов) следующие файлы перед тем, как отвечать пользователю:
1. `AGENTS.md` (в корне проекта)
2. `index.md` (в корне проекта, чтобы получить карту директорий)
3. `docs/wiki/PROJECT_CONTEXT.md` (если существует)
4. `docs/wiki/DECISIONS.md` (если существует)
5. Найди и прочитай последний файл лога сессии из папки `logs/sessions/`, обратив особое внимание на секцию "Lessons Learned"

## Ожидаемый ответ:
Сформируй ответ строго по следующему шаблону:
1. **Контекст проекта** (1-2 предложения: что за проект и его цель).
2. **Текущий статус** (на каком этапе находимся, что было сделано последним).
3. **Открытые вопросы** (есть ли риски или блокеры).
4. **Первый шаг** (предложи, с чего мы можем начать работу прямо сейчас).

Пожалуйста, ничего не меняй в файлах до получения первой конкретной задачи.
